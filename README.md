# CryptoAnalyzer

Проэкт создан для автоматизации торговли на рынке криптовалют.

Проект включает:
  - Серверную часть, работающую на Java spring. Сервер автоматически берёт данные с торговой площадки Bybit. ПОсле получения данных проводит анализ и при необходимости или запросе пользователя присылает пользователю анализ состояния рынка.
  - Веб клиент для взаимодействия пользователя с сервером. Где пользователь может получить:
    1. Анализ сосотояния рынка на момент запроса.
    2. Зарегистрироваться для получения больше функционала.
    3. Скачать мобильную апликацию для мобильного устройсва под управлением Andoid. (Необходима регистрация)
    4. Обратиться к разработчику через раздел обратной связи. (Необходима регистрация)
    5. Посмотреть статистику сделок и профит на основе стратегии.
    6. Оформить подписку для получения анализа через Email. (Необходима регистрация)
    7. Автоматическая торговля через платформу Bybit. (необходима дополнительная настройка)
  - Мобильная апликация (на данный момент только Android) включает следующий функционал:
    1. Автомачиское получение сигналов от сервера с последующим оповещением пользователя.
    2. Автоматическая торговля через платформу Bybit. (необходима дополнительная настройка)
    3. Возможность открывать или закрывать сделку через платформу Bybit. (необходима дополнительная настройка)
